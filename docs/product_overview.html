FreeRAG Local-first raw context for LLM workflows

Capture the messy context before it disappears.

FreeRAG collects local raw material on macOS. MyRAG later mines it in Codex or Claude Code and turns fragments into a matter-by-matter summary table.

English Overview 简体中文介绍

What FreeRAG Is

FreeRAG is a small native macOS menu bar app. It collects screen evidence, clipboard text/images, and voice notes into a local corpus folder. It is not a cloud knowledge base and it does not analyze your material by itself.

Collect Raw Evidence

Save screen, clipboard, and voice fragments before they get lost in app switching.

Mine With MyRAG

Use the bundled skill in Codex or Claude Code to read the local corpus and summarize it.

Clean After Confirmation

Raw material is marked done only after the user has reviewed the summary.

Why It Exists

For work that cannot be API-fed

Dashboards, ecommerce backends, chats, design references, debugging states, and meeting notes often live on screen. FreeRAG catches them locally.

For LLM workflows with evidence

MyRAG reads the collected material and produces a table with facts, risks, next steps, evidence, and confidence.

Beta Notes

  • Local corpus path: ~/Documents/Corpus/
  • Current release target: v0.5.1-beta.1
  • The beta DMG uses self-signed local stable signing and is not Apple Developer ID signed or notarized.
  • OCR and ASR are not productized inside the app yet.
