FreeRAG Local-first raw context collector + MyRAG skill

Save the context your LLM cannot see.

FreeRAG catches the screen states, clipboard fragments, screenshots, and voice notes that usually disappear before you can turn them into useful context.

How It Works

01Collect

FreeRAG stores raw screen, clipboard, and voice material locally.

02Deduplicate

Repeated clipboard images are folded by exact SHA-256 before reading.

03Mine

MyRAG reads the corpus in Codex or Claude Code and extracts matters.

04Clean

Raw material is marked done only after the user confirms the summary.

What MyRAG Produces

The useful output is not a pile of processed files. It is a compact summary table in the chat: one matter per row, with facts, numbers, people/projects, judgment, risk, next step, evidence, and confidence.

For dashboards

Extract metrics, time ranges, filters, account names, anomalies, and actionable conclusions.

For chats and meetings

Extract who said what, agreements, disagreements, commitments, deadlines, and hidden assumptions.

For messy screenshots

Keep representative evidence, skip low-value repeated crops, and mark uncertainty clearly.

Local-First Boundary

FreeRAG does not upload by itself

The app writes local raw material to ~/Documents/Corpus/. Cloud behavior only enters when the user explicitly asks an LLM tool to read that material.

Release status

The current beta uses self-signed local stable signing and is not Apple Developer ID signed or notarized. OCR and ASR are not productized inside the app yet.

What Is Included

  • Native Swift/AppKit macOS menu bar app.
  • Corpus browser with search, open, reveal, and clean-processed-raw flow.
  • DMG packaging with bundled MyRAG skill.
