FreeRAG 本地原材料收集器 + MyRAG 淘金 skill

把 LLM 看不见的上下文先留住。

FreeRAG 收住 Mac 上容易消失的屏幕状态、剪贴板碎片、截图和语音；MyRAG 再在 Codex 或 Claude Code 里把它们淘成事项表。

工作方式

01收集

FreeRAG 把屏幕、剪贴板和语音 raw 材料保存在本地。

02去重

重复剪贴板图片先按 SHA-256 精确折叠，只读代表样本。

03淘金

MyRAG 在 Codex / Claude Code 里读取语料，抽成事项。

04清理

用户看过 summary 并确认后，raw 才标记可清理。

MyRAG 产出什么

有价值的产物不是一堆 processed 文件，而是聊天窗里的高密度表格：一个事项一行，把事实、数字、人物/项目、判断、风险、下一步、证据和置信度放在同一行。

后台/看板

抽指标、时间范围、筛选条件、账号、异常点和可行动结论。

聊天/会议

抽谁说了什么、共识、分歧、承诺、deadline 和隐含假设。

混乱截图

保留代表证据，跳过低价值重复裁切，并明确标注不确定。

本地优先边界

FreeRAG 不会自行上传

App 把原材料写到 ~/Documents/Corpus/。只有用户明确让 LLM 工具读取这些材料时，才进入对应 LLM 工具的隐私边界。

Beta 状态

当前包是 ad-hoc signed，尚未公证。OCR 和 ASR 还没有产品化进 app。

包含内容

  • Swift/AppKit 原生 macOS 菜单栏 app。
  • 原材料库搜索、打开、Finder 定位和清理已处理 raw。
  • DMG 打包，附带 MyRAG skill。
  • MIT 开源。
