# FreeRAG Docs

当前文档只保留产品交付相关内容：

- `PRODUCT_PRD.md`：产品定义、范围边界、当前符合度和下一步。
- `DESIGN_GUIDE.md`：FreeRAG 的原生 Mac 设计规则。
- `QUALITY_BAR.md`：测试前必须满足的质量标准。
- `product_overview.html`：双语入口产品说明页。
- `product_overview.en.html`：英文产品说明页。
- `product_overview.zh-CN.html`：简体中文产品说明页。
- `PROJECT_STATUS.md`：公开发布状态摘要。

`CURRENT_STATE.md` 是本地交接文件，可能包含机器路径和会话细节，不进入公开仓库。
